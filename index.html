API

Developer API

Integrate SpamShield AI into your applications with our simple REST API.

POST /predict

Analyze email content and return spam prediction, probability, confidence, risk level, and detected keywords.

Request

curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{"text": "CONGRATULATIONS! You have won $1,000,000! Click here to claim your prize now!"}'

Response

{
  "prediction": "Spam",
  "probability": 0.95,
  "confidence": 95,
  "risk": "Critical",
  "keywords": ["congratulations", "won", "claim", "click here", "prize"]
}
POST /predict

JavaScript Example

const response = await fetch("http://localhost:5000/predict", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ text: "Your email content here" })
});

const result = await response.json();
console.log(result.prediction);  // "Spam" or "Ham"
console.log(result.probability); // 0.0 - 1.0
console.log(result.confidence);  // 0 - 100
console.log(result.risk);         // "Low", "Medium", "High", "Critical"
console.log(result.keywords);     // ["suspicious", "words", ...]

Response Fields

prediction
"Spam" or "Ham"
probability
0.0 - 1.0 float
confidence
0 - 100 integer
risk
Low/Medium/High/Critical
keywords
Array of detected suspicious keywords
About

The AI Engine

SpamShield AI combines machine learning, heuristic analysis, and cybersecurity expertise to deliver enterprise-grade email protection.

Machine Learning

Our engine uses TF-IDF vectorization to convert email text into numerical features, then applies a Multinomial Naive Bayes classifier trained on thousands of labeled spam and ham emails to predict whether new emails are malicious.

Cyber Security

We combine ML with rule-based heuristics that detect phishing patterns, urgency tactics, financial lures, sender domain anomalies, and content formatting tricks that spammers use to evade traditional filters.

AI Engine

The SpamShield AI engine processes emails through 20+ detection rules across 8 categories, producing a weighted threat score from 0 to 100 with a transparent breakdown of every triggered indicator.
